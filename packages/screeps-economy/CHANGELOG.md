## [0.1.0] - 2024-12-30

### Added
- Initial extraction from screeps-ant-swarm bot
- LinkManager for automated energy transfers
- TerminalManager for inter-room routing with Dijkstra optimization
- FactoryManager for commodity production
- MarketManager for trading with price analysis
- Emergency resource transfer support
- Comprehensive test suite
- Full TypeScript type definitions
