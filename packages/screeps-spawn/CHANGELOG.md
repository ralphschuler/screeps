## [0.1.0] - 2024-12-30

### Added
- Initial extraction from screeps-ant-swarm bot
- SpawnManager class for spawn coordination
- Body optimization system
- Priority-based spawn queue
- Role templates for economy, military, utility, and power roles
- Pheromone-based priority adjustments
- Bootstrap mode support
- Comprehensive test suite
- Full TypeScript type definitions
